kinect-ultra V1.0a
Author: Dark Best
Release Date: 4/23/2017

Qu'est-ce que c'est?

Ceci est un Hack Kinect pour vous transformer en super-héros et utiliser
ses superpuissances.

1. Préalable

(1) Vous avez déjà Kinect, n'est-ce pas?
(2) Ce produit est censé fonctionner sur plusieurs plates-formes Windows bien que
    il est seulement testé sur Windows XP Pro SP3 32 bits et Windows 7 Pro 64 bits
    à ce moment là.
(3) Votre ordinateur devrait avoir un processeur et un GPU suffisamment rapides.
    - L'exigence minimale est inconnue
    - Le GPU doit prendre en charge OpenGL 2.x ou supérieur, les shaders programmables,
      et les VAO au moins.
    - Core i7 2600 + GeForce GTX 285 est suffisant au moins.

2. Installation

2.1 Pour la version Kinect SDK

(1) Si vous ne l'avez pas encore fait, téléchargez le kit SDK Kinect et installez-le.
    http://www.kinectforwindows.org

(2) Déballez cette archive où vous voulez.

2.2 Pour la version OpenNI

(1) Si vous ne l'avez pas encore fait, téléchargez OpenNI et d'autres connexes
    logiciel et installez-les. La procédure change souvent, alors voyez le
    informations à l'URL ci-dessous.
    http://code.google.com/p/kinect-ultra/wiki/OpenNI_Downloads

(2) Déballez cette archive où vous voulez.

3. Comment utiliser

(1) Il suffit d'exécuter kinect-ultra.exe.
    - Définissez le répertoire de travail au même emplacement que l'exécutable.
    - Appuyez sur la touche [ESC] pour terminer.
    - Appuyez sur la touche [Entrée] pour l'afficher en plein écran.
